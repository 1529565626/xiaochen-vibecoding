# my-net-disk · 局域网文件共享工具

在电脑上运行，手机 / 平板 / 其他电脑通过浏览器访问，局域网内传文件。

---

## 使用前准备

电脑需要安装 **Node.js**（运行环境，装一次即可）。

1. 打开 [https://nodejs.org](https://nodejs.org)
2. 下载 **LTS（长期支持版）**，比如 `v20.x.x LTS`
3. 双击安装，一路点"下一步"完成

> 装过了？跳过这一步。

---

## 如何使用

1. 解压 `my-net-disk-release.zip`
2. 双击 `start.bat`
3. 浏览器会自动打开管理页面
4. 右下角二维码点开，手机扫码就能访问

---

## 管理页面能做什么

打开后看到的是**管理端**，可以：

- **文件管理**：查看文件、新建文件夹、重命名、删除
- **修改配置**：改端口、改存储位置、调整上传大小限制
- **打开本地目录**：一键跳转到电脑上的文件夹
- **查看二维码**：右下角按钮，手机扫码后进入客户端页面

## 其他设备怎么用

手机或平板连上**同一个 WiFi**，扫码后自动打开客户端页面，可以：

- 浏览文件，点击下载
- 上传文件到电脑

---

## 常用地址

| 地址 | 用途 |
|------|------|
| `http://127.0.0.1:3000` | 本机打开管理端 |
| `http://192.168.x.x:3000` | 局域网内其他设备访问（扫码得到的地址） |

> `192.168.x.x` 是电脑的局域网 IP，启动后命令行会显示。

---

## 修改存储位置

默认文件存在 `E:\my-netdisk\files\` 下。想换个盘？

1. 打开管理端 → 点击左侧「系统配置」
2. 修改「下载目录」「上传临时目录」「临时目录」三个路径
3. 点「保存配置」
4. 关闭命令行窗口，重新双击 `start.bat` 生效

---

## 关闭服务

关掉命令行窗口即可。

---

## 常见问题

**Q: 手机扫了码打不开？**
检查手机和电脑是否连在同一个 WiFi。公司网络有隔离策略的话可能不通，可以电脑开热点，手机连热点。

**Q: 想换个端口？**
管理端 → 系统配置 → 修改端口号 → 保存 → 重启。

**Q: 上传文件有大小限制吗？**
默认单个文件最大 10GB。可以在系统配置里调整。
