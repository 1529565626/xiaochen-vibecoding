# xiaochen-vibecoding

关于一些vibecoding学习，想法，场景实现的案例记录



## 01-net-disk

局域网网盘小应用，能快速进行移动端设备、电脑端设备文件上传下载管理。理论上能够拉满路由器的上传下载速度（受硬盘写入速度和路由器的荷载速度限制）。电脑端运行，移动端扫码（浏览器自带）打开访问，进行文件管理。
