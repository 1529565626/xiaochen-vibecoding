# 飞书机器人搭建与接入指南

从零开始，一步步创建飞书机器人并接入 feishu-claude-bot。

## 一、创建飞书应用

1. 打开 [飞书开发者后台](https://open.feishu.cn)，登录你的飞书账号
2. 点击「创建应用」→ 选择「**企业自建应用**」
3. 填写应用名称（如「Claude 助手」）和描述
4. 创建完成后进入应用详情页

## 二、获取凭证

在应用详情页左侧「**凭证与基础信息**」：

- 复制 **App ID** → 填入 `.env` 的 `FEISHU_APP_ID`
- 复制 **App Secret**（点「显示」）→ 填入 `.env` 的 `FEISHU_APP_SECRET`

## 三、配置事件订阅

左侧「**事件与回调**」（或叫「事件订阅」）：

1. 模式选择「**长连接**」（不要选 Webhook）
2. 页面会出现 **Encrypt Key** 和 **Verification Token**，分别填入 `.env`：
   - `FEISHU_ENCRYPT_KEY`
   - `FEISHU_VERIFICATION_TOKEN`
3. 在「事件订阅」区域，点击「添加事件」→ 搜索并勾选：
   - **接收消息** `im.message_receive_v1`
4. 保存

## 四、启用机器人

左侧「**应用功能**」→「机器人」→ 开启。

可选配置：
- 机器人名称
- 机器人头像
- 机器人简介（如「我是 AI 助手，可以帮你处理各种任务」）

## 五、添加权限

左侧「**权限管理**」→ 确保以下权限已添加：

| 权限 | 用途 |
|---|---|
| `im:message` | 读取消息 |
| `im:message:send_as_bot` | 以机器人身份发送消息 |
| `im:chat` | 获取群聊信息 |

> 长连接模式下，大部分权限会自动添加。如果后续提示权限不足，来这里补充。

## 六、发布应用

1. 右上角点击「**创建版本**」→ 填写版本号（如 `1.0.0`）和更新说明
2. 保存后点击「**申请发布**」
3. 飞书企业管理员在后台审批（如果你是管理员，会自动通过，或去管理后台手动审批）
4. 状态变为「**已发布**」即完成

## 七、添加到群聊

1. 打开目标飞书群
2. 右上角 `…` →「设置」→「群机器人」
3. 点击「添加机器人」→ 搜索你的应用名称 → 添加
4. 群内出现「xxx 加入群聊」提示

> 注意：是添加「企业自建应用」机器人，不是「自定义群机器人」（Webhook 那个）。

## 八、验证

启动服务：

```bash
cd feishu-claude-bot
python main.py
```

看到日志：

```
✅ 飞书长连接已建立，等待消息...
```

然后在群里 @机器人 发一条「你好」测试。

## 常见问题

**Q: 机器人收到消息没反应？**
- 确认应用已发布（状态为「已发布」）
- 确认长连接模式已开启
- 确认 `im.message_receive_v1` 事件已订阅

**Q: 提示"待上线"？**
执行第六步，创建版本并申请发布。

**Q: 找不到 Encrypt Key 和 Verification Token？**
确保事件订阅模式选择了「长连接」。如果选了 Webhook 模式则显示的是回调地址，不是这两个字段。

**Q: 群聊里搜不到机器人？**
确认机器人已发布且已开启机器人功能。发布后等 1-2 分钟刷新群聊设置。
